# Introduction to Programming & Advanced Course in Programming 2026

Welcome to the official course material for the Department of Computer Science at the **University of Helsinki**. This repository contains the curriculum and exercise structure for the following modules:

  * **BSCS1001: Introduction to Programming** (5 ECTS) — *Parts 1 through 7*
  * **BSCS1002: Advanced Course in Programming** (5 ECTS) — *Parts 8 through 14*

-----

## 🗓️ Important Dates

  * **Course Start:** January 12th, 2026
  * **Course End:** December 31st, 2026
  * **Next Lecture Series:** Autumn 2026

-----

## 🎓 Requirements for Completion

To pass either course, you must fulfill the following criteria:

1.  **Programming Exercises:** Complete the required exercises for each part.
2.  **Examination:** Participate in the final exam.

> [\!IMPORTANT]
> Detailed grading scales and exam schedules can be found on the **Grading and Exams** page.
